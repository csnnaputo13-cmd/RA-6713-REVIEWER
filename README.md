📖 Overview
A comprehensive, interactive reviewer for Republic Act No. 6713 – the Code of Conduct and Ethical Standards for Public Officials and Employees. This web-based application features the full text of the law, detailed question banks for each section, and an intelligent analysis system that breaks down every answer with step-by-step explanations.

🚀 Getting Started
Prerequisites
Any modern web browser (Chrome, Firefox, Safari, Edge)

Internet connection (for text-to-speech voices)

Installation
Simply download the CSNRA6713.html file and open it in your web browser.

🎮 How to Use
Welcome Screen
Enter your name

Choose your mode:

📖 Review First – Read the full text of RA 6713

📝 Take Test – Answer practice questions

Review Mode
📄 Read the complete text of RA 6713

🔊 Tap the "Read Full Text" button for audio narration

🎙️ Choose from available text-to-speech voices

⏱️ Adjust reading speed with the slider

🖨️ Print the law text for offline study

Test Mode
Select a section from the dropdown menu

Answer each question by tapping/clicking your choice

Click "Submit Answer" to reveal the analysis

Review the formatted breakdown:

📌 STEP-BY-STEP ANALYSIS

📖 WORD DEFINITION

🏛️ CONTEXT ANALYSIS

✅ CONCLUSION

📚 REFERENCES

Navigate with "PREV" and "NEXT" buttons

On the last question, click "Submit Exam" for results
